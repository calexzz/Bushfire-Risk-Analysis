# Анализ влияния гидрометеорологических условий на риск лесных пожаров

Репозиторий содержит аналитическое исследование и комплексную предобработку гидрометеорологических данных для оценки и прогнозирования пожарного риска Иркутской области.

## Ссылки на исходные файлы и ресурсы

* Репозиторий проекта: [ссылка на GitHub](https://github.com/calexzz/Bushfire-Risk-Analysis)
* Аналитический блокнот: [ссылка на Jupyter Notebook](https://github.com/calexzz/Bushfire-Risk-Analysis/blob/master/bushfire_analysis_final.ipynb)
* Исходные данные посуточных наблюдений (ИКИ РАН): [ссылка на датасет](https://disk.yandex.ru/d/EkgRoEGBOWyQaw)
* Эталонные данные температуры (CTSS): [ссылка на файл .nc](https://zenodo.org/record/5594045)
* Эталонные данные осадков (CRSS): [ссылка на файл .nc](https://zenodo.org/record/4472614)

## Основные задачи и этапы работы

* Верификация и коррекция данных:
  * Загрузка и предобработка исходных данных посуточных наблюдений температуры и осадков за период 2004–2024 гг.
  * Интерполяция и приведение данных к регулярной эталонной сетке 0.25° за период 1979–2020 гг.
  * Оценка валидности исходных датасетов по ключевым метрикам на пересекающемся временном интервале.
* Моделирование и анализ факторов риска:
  * Построение и сравнение базовой и скорректированной моделей машинного обучения для оценки вероятности возникновения лесных пожаров.
  * Анализ и визуализация важности признаков (feature importance) для определения ключевых гидрометеорологических драйверов пожарной активности.

## Используемый стек технологий

* Язык программирования: Python
* Анализ данных и вычисления: pandas, numpy, xarray, scipy, geopandas
* Визуализация: matplotlib, seaborn
* Среда разработки: Jupyter Notebook
